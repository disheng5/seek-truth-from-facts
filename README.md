# 🧭 Seek Truth from Facts（实事求是）

面向业务决策者的东方哲学方法论 Skill，基于毛泽东选集（1-5卷）核心思想，提取哲学方法论应用于现代商业决策。

## 核心理念

基于"活的灵魂"三大原则：

- **实事求是** — 一切从客观实际出发，在事物内部联系中找规律
- **群众路线** — 从群众中来、到群众中去，好决策来自一线洞察
- **独立自主** — 立足自身力量，善于学习借鉴，绝不照搬照抄

## 八大方法论框架

| # | 方法论 | 核心能力 | 典型场景 |
|:---|:---|:---|:---|
| 1 | **矛盾分析法** | 识别主要矛盾，纲举目张 | 复杂问题诊断、优先级决策、资源分配 |
| 2 | **实践认识论** | 实践→认识→再实践，螺旋上升 | 产品迭代、方案验证、经验复盘 |
| 3 | **持久战略法** | 阶段划分，以时间换空间 | 长期规划、市场竞争、逆境突围 |
| 4 | **调查研究法** | 没有调查就没有发言权 | 市场分析、用户研究、立项论证 |
| 5 | **群众路线法** | 从群众中来，到群众中去 | 团队管理、需求洞察、组织变革 |
| 6 | **统一战线法** | 团结一切可以团结的力量 | 跨部门协作、资源整合、生态合作 |
| 7 | **独立自主法** | 自力更生为主，争取外援为辅 | 技术路线选择、核心能力建设 |
| 8 | **批评与自我批评法** | 勇于正视问题，持续改进 | 项目复盘、团队冲突、文化建设 |

## 特色

- **问题前置分类**：自动识别实践型/认识型/准认识型/边界模糊型问题
- **三种交互模式**：问题诊断（最常用）、指定框架、综合战略
- **实践检验**：每次分析输出可验证假设 + 验证方法 + 成功标准 + 失败应对
- **批判性反思**：核心假设审视 + 可能反驳 + 分析局限，保持智识诚实
- **经典引用规范**：引用嵌入论证，必须展开分析，杜绝装饰性引用
- **统筹兼顾**：多目标场景不简单做减法，支持权重分配与平衡

## 使用方式

### 作为 Claude Code / Mira Skill

将 `SKILL.md` 放入你的 Skill 目录即可自动加载。

### 触发示例

- "帮我分析一下当前的竞争形势"
- "团队有分歧怎么办"
- "这个项目该怎么规划"
- "我们应该自研还是外购"
- "项目失败了怎么复盘"
- "资源不够，怎么排优先级"
